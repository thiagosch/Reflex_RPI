# gphoto2 python library installation
System wide requirements
  
- libexif12
- libgphoto2-6
- libgphoto2-port12
- libltdl7
- libgphoto2-dev
- gphoto2


`sudo apt install libexif12 libgphoto2-6 libgphoto2-port12 libltdl7 libgphoto2-dev gphoto2`
